> Trigger: When this folder's structure/responsibilities/file list changes, update this document.

Docker contains runtime configuration for the single-server deployment topology. / Docker 包含单机部署拓扑的运行配置。
Nginx serves static H5 assets and proxies backend PHP requests. / Nginx 提供静态 H5 资源并代理后端 PHP 请求。
Runtime config must preserve `/api` and `/sanctum` as Laravel-owned paths. / 运行配置必须保持 `/api` 与 `/sanctum` 归 Laravel 所有。

## Files
- `nginx/`: docker/nginx - reverse proxy and static caching config / 反向代理与静态缓存配置
- `php/`: docker/php - PHP-FPM image config / PHP-FPM 镜像配置
