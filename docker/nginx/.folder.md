> Trigger: When this folder's structure/responsibilities/file list changes, update this document.

Nginx owns static frontend delivery, direct public-disk upload delivery, and Laravel reverse proxy routing. / Nginx 负责前端静态交付、直接 public disk 上传资源交付与 Laravel 反向代理路由。
`/api`, `/admin`, `/sanctum`, and Filament/Livewire assets go to backend; `/storage` aliases the public disk. / `/api`、`/admin`、`/sanctum` 与 Filament/Livewire 资源进入后端；`/storage` 直接 alias 到 public disk。
Hashed assets may be cached long-term; HTML must stay fresh. / 带 hash 的资源可长期缓存；HTML 必须保持新鲜。
