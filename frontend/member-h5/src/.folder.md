Vue source is split by API, state, route, view, component, type, utility, and mobile shell boundaries. / Vue 源码按 API、状态、路由、视图、组件、类型、工具与移动端外壳边界拆分。
Stores own registration math; views only render and dispatch user actions. / Store 拥有报名计算；视图只渲染并派发用户动作。
Submit payloads must be derived from local state and never include trusted total amounts. / 提交数据必须由本地状态派生，且不包含可信总金额。
