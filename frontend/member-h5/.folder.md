This app gives members a phone-first registration workflow with local matrix state. / 本应用提供手机优先的会员报名流程与本地矩阵状态。
It calls backend only for login, race list, bootstrap, submit, detail, and logout. / 它只在登录、赛事列表、初始化、提交、详情和退出时访问后端。
Every click in single and multi registration must update local state without API traffic. / 单羽与多羽报名的每次点击都必须只更新本地状态，不触发 API。
