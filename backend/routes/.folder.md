# Folder: backend/routes

> Trigger: When this folder's structure/responsibilities/file list changes, update this document.
> 触发条件:当本文件夹的结构/职责/文件列表变化时，更新此文档。

Routes expose no-store member API, profile/password API, registration history API, published race detail API, public information API, public storage, protected ring-sale receipts, web entry, and scheduler boundaries. / 路由暴露 no-store 会员 API、档案改密 API、报名历史 API、已发布赛事明细 API、公开信息 API、公开存储、受保护售环收据、Web 入口与调度边界。
Member API routes use web sessions but must authenticate profile and business routes with the `member` guard. / 会员 API 路由使用 Web 会话但档案与业务路由必须通过 `member` guard 鉴权。
No route should perform business validation inline or block login-based account switching. / 任何路由都不应内联业务校验或阻断通过登录切换账号。

## Files
- `api.php`: backend/routes - public information routes and no-store member-guard API routes including registration history and published race details / 公开信息路由与包含报名历史、已发布赛事明细的 no-store member guard 会员 API 路由
- `console.php`: backend/routes - scheduler definitions / 调度定义
- `web.php`: backend/routes - web, public storage, and permission-protected ring-sale receipt routes / Web、公开存储与受权限保护的售环收据路由
