# Folder: backend/public

> Trigger: When this folder's structure/responsibilities/file list changes, update this document.
> 触发条件:当本文件夹的结构/职责/文件列表变化时，更新此文档。

Public contains the Laravel HTTP front controller, published admin assets, PWA files, and storage symlink target. / public 包含 Laravel HTTP 前端控制器、已发布后台资源、PWA 文件与存储软链接目标。
Nginx routes PHP-owned paths, Filament assets, and `/storage` uploads here. / Nginx 将 PHP 归属路径、Filament 资源与 `/storage` 上传资源路由到这里。
No application secrets or writable runtime state belong here. / 这里不应放应用密钥或可写运行状态。

## Files
- `index.php`: backend/public - Laravel HTTP entrypoint / Laravel HTTP 入口
- `admin/`: backend/public/admin - admin PWA manifest, service worker, and PNG/SVG installation icons / 后台 PWA 清单、Service Worker 与 PNG/SVG 安装图标
- `css/`: backend/public - published Filament CSS / 已发布的 Filament CSS
- `fonts/`: backend/public - published Filament fonts / 已发布的 Filament 字体
- `js/`: backend/public - published Filament JavaScript / 已发布的 Filament JavaScript
- `storage/`: backend/public - runtime symlink to public uploaded files / 指向公开上传文件的运行时软链接
