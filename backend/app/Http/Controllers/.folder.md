Controllers translate HTTP into service calls or storage responses. / 控制器把 HTTP 翻译为服务调用或存储响应。
They must not own registration rules. / 它们不得拥有报名规则。
Member API controllers live under the API subtree; public storage is a root web controller. / 会员 API 控制器位于 API 子树；公开存储是根级 Web 控制器。
