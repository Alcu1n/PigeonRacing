Member controllers expose branding, forced-switch login, profile/password, race bootstrap, published race details, submit, detail, and logout flows. / 会员控制器暴露品牌、强制切换登录、档案改密、赛事初始化、已发布赛事明细、提交、详情与退出流程。
Bootstrap is cached; submit is transactional and service-owned. / 初始化可缓存；提交由服务层事务负责。
Members can only read their own profile, pigeons, and registrations. / 会员只能读取自己的档案、足环与报名记录。
