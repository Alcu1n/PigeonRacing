# Folder: backend/app/Providers

> Trigger: When this folder's structure/responsibilities/file list changes, update this document.
> 触发条件:当本文件夹的结构/职责/文件列表变化时，更新此文档。

Providers bind application-wide Laravel and Filament boot behavior. / 提供者绑定应用级 Laravel 与 Filament 启动行为。
They should register infrastructure, not hide business workflows. / 它们应注册基础设施，不隐藏业务流程。
Admin panel setup belongs under the Filament provider subtree. / 后台面板设置属于 Filament 提供者子树。

## Files
- `AppServiceProvider.php`: backend/app/Providers - application service provider / 应用服务提供者
- `Filament/`: backend/app/Providers/Filament - Filament panel providers / Filament 面板提供者
