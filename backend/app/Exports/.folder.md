# Folder: backend/app/Exports

> Trigger: When this folder's structure/responsibilities/file list changes, update this document.
> 触发条件:当本文件夹的结构/职责/文件列表变化时，更新此文档。

Exports convert trusted backend data into Excel workbooks. / 导出类把可信后端数据转换为 Excel 工作簿。
They shape rows and headings only; multi-group exports must emit one row per unique group. / 它们只塑造行与表头；多羽导出必须按唯一组合每组一行。
Workbook columns must match the admin workflows documented in README. / 工作簿列必须匹配 README 记录的后台流程。

- `PigeonImportErrorExport.php`: backend/app/Exports - failed pigeon import row report / 足环导入失败行报告
- `PigeonImportTemplateExport.php`: backend/app/Exports - pigeon import template workbook / 足环导入模板表格
- `RegistrationMatrixExport.php`: backend/app/Exports - race registration matrix with single rows and unique multi-group rows / 单羽行与唯一多羽组合行的赛事报名矩阵导出
