# Folder: backend/app/Exports

> Trigger: When this folder's structure/responsibilities/file list changes, update this document.
> 触发条件:当本文件夹的结构/职责/文件列表变化时，更新此文档。

Exports convert trusted backend data into Excel workbooks. / 导出类把可信后端数据转换为 Excel 工作簿。
They shape rows and headings only; business validation stays in services. / 它们只塑造行与表头；业务校验留在服务层。
Workbook columns must match the admin workflows documented in README. / 工作簿列必须匹配 README 记录的后台流程。
