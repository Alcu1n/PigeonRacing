# Folder: backend/app/Filament

> Trigger: When this folder's structure/responsibilities/file list changes, update this document.

Filament owns administrator permission management, ring-sale operations, review, progressive category, branding, and operational surfaces. / Filament 负责管理员权限管理、售环运营、审核、递进类别、品牌与运营界面。
It edits configuration and public assets but must not bypass service-owned registration rules. / 它编辑配置与公开资源，但不得绕过服务层报名规则。
Admin changes that affect member bootstrap must clear or version caches. / 影响会员初始化的后台变更必须清缓存或递增版本。

## Files
- `Pages/`: backend/app/Filament/Pages - singleton admin settings pages / 单例后台设置页面
- `Auth/`: backend/app/Filament/Auth - administrator authentication pages / 管理员鉴权页面
- `Concerns/`: backend/app/Filament/Concerns - shared resource authorization adapters / 共享资源授权适配器
- `Clusters/`: backend/app/Filament/Clusters - grouped admin modules with one sidebar entry and top sub-navigation / 使用单一侧栏入口与顶部子导航的后台模块组
- `Resources/`: backend/app/Filament/Resources - admin resources for members, ring sales, races, projects, progressive categories, pigeons, registrations, and information posts / 会员、售环、赛事、项目、递进类别、足环、报名与信息发布后台资源
