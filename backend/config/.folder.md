# Folder: backend/config

> Trigger: When this folder's structure/responsibilities/file list changes, update this document.
> 触发条件:当本文件夹的结构/职责/文件列表变化时，更新此文档。

Config files bind Laravel runtime to environment-specific services. / 配置文件把 Laravel 运行时绑定到环境服务。
Auth and Sanctum preserve same-origin and LAN-host cookie member sessions. / 鉴权与 Sanctum 保持同源和局域网 Host 的 Cookie 会员会话。
Database, cache, session, and public storage defaults must match Docker, LAN, tunnel, and domain access. / 数据库、缓存、会话与公开存储默认值必须匹配 Docker、局域网、隧道与域名访问。

## Files
- `app.php`: backend/config - app identity and locale / 应用身份与语言
- `auth.php`: backend/config - admin and member guards / 管理员与会员守卫
- `cache.php`: backend/config - cache stores / 缓存存储
- `database.php`: backend/config - MySQL and Redis connections / MySQL 与 Redis 连接
- `filesystems.php`: backend/config - local disks and same-origin public storage URLs / 本地磁盘与同源公开存储 URL
- `sanctum.php`: backend/config - SPA cookie auth config / SPA Cookie 鉴权配置
- `session.php`: backend/config - session cookie and store config / 会话 Cookie 与存储配置
